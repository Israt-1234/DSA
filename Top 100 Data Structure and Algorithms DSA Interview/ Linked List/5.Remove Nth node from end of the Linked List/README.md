# Question:

Given a linked list. The task is to remove the Nth node from the end of the linked list.
---

### Example:
**Input:**  
LinkedList = 1 ->2 ->3 ->4 ->5 , N = 2 <br>
**Output:**  
1 ->2 ->3 ->5

---

### Explanation: <br>
Linked list after deleting the 2nd node from last which is 4, is 1 ->2 ->3 ->5
